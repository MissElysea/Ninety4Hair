## Steps

1. Install Gulp Globally

```
npm i -g gulp-cli
```

2. Start the dev server

```
npm run dev
```

or

```
yarn dev
```

3. Start the dev server with proxy

```
npm run proxy
```

4. Build files for production

```
yarn build
```

or

```
npm run build
```

5. Deploy for production script

```
npm start
```

or

```
yarn start
```
